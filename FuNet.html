Instructions for use:

Offline use: 1. Click to download the Python program we provided, you will get the genelist (input.txt) containing the python executable program, FGSG database, test data; 2. Double-click findmodule.py to execute input.txt and get output.txt. 3. By looking at output.txt, you can see which module each gene belongs to, and in the last line, you can see the most enriched module of the gene. So you can go to the targeted research module and the main regulators in the module.

Online use: 1. Select your genelist as input, you can import a file or type in the input box; 2. Select your database as the search database. When you study FGSG, you can directly download the FGSG database we provide as input. 3. After the input is completed, click run to execute, and the database and search results will be displayed below. You can see the modules to which each gene belongs and display the most enriched module in the last row. You will be able to conduct targeted research, the main regulators within modules and modules.
